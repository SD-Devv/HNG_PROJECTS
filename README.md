## Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/SD-Devv/HNG_PROJECTS.git
   ```
2. Install dependencies:
   ```bash
   cd hng-public-api
   npm install
   ```
3. Run the app:
   ```bash
   node index.js
   ```
